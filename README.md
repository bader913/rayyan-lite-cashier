# Rayyan Lite / CashierApp Foundation

برنامج كاشير خفيف مبني بـ Python + PySide6 + SQLite، منفصل تماماً عن Rayyan Pro.

## الهدف من النسخة الأولى

- شاشة منتجات بسيطة.
- شاشة بيع سريع POS.
- حفظ الفواتير داخل SQLite.
- تحديث المخزون داخل transaction.
- تسجيل حركة مخزون لكل بيع أو تعديل رصيد.
- حسابات مالية باستخدام `Decimal` وليس `float`.
- نسخ احتياطي ZIP من داخل البرنامج.

## التشغيل

من داخل مجلد المشروع:

```powershell
cd "C:\Users\Bader\OneDrive\Desktop\CashierApp"
Set-ExecutionPolicy -Scope Process Bypass -Force
.\run.ps1
```

## مكان قاعدة البيانات

البرنامج ينشئ قاعدة البيانات هنا تلقائياً:

```text
data\cashier_lite.sqlite
```

## ملاحظات أمان

- لا توجد أسرار داخل المشروع.
- لا يوجد اتصال خارجي.
- كل عملية بيع تتم داخل transaction.
- لا يتم استخدام float في الحسابات المالية داخل الخدمات الأساسية.


## تحديث UI / العملة / الفواتير الحرارية

- تحسين ثيم الواجهة والخطوط والجداول.
- إضافة إعدادات العملة الأساسية ورمزها وعملة المقارنة وسعر الصرف.
- جعل آخر الفواتير في لوحة اليوم قابلة للفتح بالضغط المزدوج.
- بعد حفظ فاتورة البيع يتم فتح نافذة تفاصيل الفاتورة مباشرة.
- إضافة معاينة HTML وطباعة حرارية 80mm من نافذة الفاتورة.


## تحسينات الواجهة والنسخ الاحتياطي

- دعم سمة فاتحة وداكنة من الإعدادات.
- تصدير واستيراد نسخة احتياطية ZIP.
- مسح البيانات التجارية مع نسخة احتياطية تلقائية قبل التنفيذ.

## تحسينات سلة البيع بالكيبورد

- يمكن تعديل الكمية والسعر والخصم مباشرة من جدول السلة.
- السعر في السلة قابل للتغيير قبل حفظ الفاتورة.
- `Enter` ينتقل بين خانات الكمية والسعر والخصم.
- `F2` يفتح تعديل الخانة الحالية.
- `Delete` يحذف السطر المحدد من السلة.
- `Ctrl+S` يحفظ الفاتورة.
- `F3` يرجع التركيز إلى خانة البحث.


## صفحة المشتريات

- إضافة صفحة **المشتريات** لإدخال البضاعة بشكل نظامي بدل تعديل المخزون اليدوي.
- يمكن البحث عن المنتج وإضافته لفاتورة شراء.
- يمكن تعديل الكمية وسعر الشراء من جدول الفاتورة مباشرة.
- حفظ فاتورة المشتريات يزيد المخزون داخل transaction ويسجل حركة `purchase` في `stock_movements`.
- يوجد خيار لتحديث سعر الشراء في بطاقة المنتج من آخر فاتورة مشتريات.
- يتم عرض آخر فواتير المشتريات من نفس الصفحة.
